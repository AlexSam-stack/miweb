Semana 1: Fundamentos Web

Temas principales

  • Conceptos: web, app, sitio, página
  • Tecnologías básicas: HTML, CSS
  • Cómo funciona la web: DNS, TCP/IP, HTTP
  • Roles web: Frontend, Backend, Fullstack
  • Herramienta: Visual Studio Code

Ejercicio

Identifica y explica la diferencia entre sitio web, página web y aplicación web.
Realiza un “Hola Mundo” en HTML y CSS en VS Code.

Código de ejemplo


<!DOCTYPE html>
<html lang="es">
<head>
    <meta charset="UTF-8">
    <title>Hola Mundo Web</title>
    <style>
        body { font-family: Arial, sans-serif; background: #f8fafc; }
        h1 { color: #1976d2; }
        .ejemplo { color: #388e3c; }
    </style>
</head>
<body>
    <h1>¡Hola Mundo!</h1>
    <p class="ejemplo">Esto es una página web simple usando HTML y CSS.</p>
</body>
</html>

Resultado esperado del código:

¡Hola Mundo!

Esto es una página web simple usando HTML y CSS.

Reflexión

Comprendí la diferencia entre los conceptos web y cómo se conectan los distintos roles y tecnologías para crear experiencias digitales.
